# Movie Review Sentiment Analyzer

A simple web app that uses Hugging Face Transformers to predict if a movie review is Positive or Negative.

## Live Demo
https://div-movie-sentiment-analyzer.streamlit.app/

## Technologies
- Python
- Streamlit
- Hugging Face Transformers (sentiment-analysis pipeline)
- PyTorch

## How to run locally
1. Clone the repo
2. `pip install -r requirements.txt`
3. `streamlit run app.py`
